# Bootstrap
- 오픈 소스인 Bootstrap.css를 불러오고, 해당 파일에 정의된 버튼, 레이아둣등을 가져와서 사용
- 클래스로 간단하게 불러와서 사용
- 트위터에서 개발

#

### 레이아웃
부트스트랩 레이아웃은 총합 12로 구성되어 있음.
- 768px 보다 클 때 : col-sm-
- 970px 보다 클 때 : col-md-
- 1170px 보다 클 때 : col-lg-
- 768px 보다 작을 때 : col-xs-
